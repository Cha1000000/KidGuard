# Веха 5 — VPN-блокировка интернета (черновик)

> ⚠️ **ЧЕРНОВИК.** Самый технически сложный модуль. **Перед стартом вехи** пересмотреть, выбрать
> уровень фильтрации и обсудить нерешённое.

Обзор всех вех — `roadmap.md`. Зависит от **вех 2 и 4** (состояние «время вышло» и адрес сервера
для whitelist). Идёт после того, как остальное работает.

## Цель вехи

Блокировать доступ ко всем внешним ресурсам после истечения лимита — на Wi-Fi и мобильном
интернете — **кроме связи с нашим сервером** (иначе синхронизация не работает).

## Ключевое ограничение

Без Device Owner **нельзя программно** форсить always-on VPN. Поэтому при первичной настройке
родитель вручную включает в системных настройках «Always-on VPN» + «Блокировать соединения без
VPN» для KidGuard (шаг мастера из вехи 1 напоминает об этом).

## Декомпозиция на шаги

### Шаг 5.1 — Скелет VpnService
- Локальный `VpnService`, consent-диалог, тестовый перехват трафика (tun-интерфейс).
- **Критерий:** VPN поднимается, трафик проходит через наш сервис.

### Шаг 5.2 — Режимы работы
- «Время есть» → VPN прозрачен (или выключен). «Время вышло» → режем весь трафик.
- Связка с состоянием лимита из вехи 2.
- **Критерий:** переключение режима по состоянию лимита работает.

### Шаг 5.3 — Whitelist сервера
- Пропускать трафик к нашему серверу (по IP/домену) даже в режиме блокировки — чтобы синк жил.
- **Критерий:** в режиме блокировки внешние сайты недоступны, связь с сервером сохраняется.

### Шаг 5.4 — Always-on + lockdown
- Инструкция и проверка, что always-on/lockdown включены (настраивается вручную при первичной
  настройке; программно без Device Owner нельзя).
- **Критерий:** при активном always-on трафик без VPN не утекает.

## Definition of Done

- После истечения лимита интернет заблокирован на Wi-Fi и мобильном.
- Связь с сервером (синхронизация) сохраняется.
- always-on/lockdown настроены; обход через отключение VPN затруднён.

## Открытые вопросы / решить в начале вехи

- **Уровень фильтрации:** DNS-based (проще) vs полноценный разбор пакетов (надёжнее) — выбрать.
- Совместимость VpnService с блокировкой интернета: не ломает ли синхронизацию и белый список.
- Поведение при переключении Wi-Fi ↔ мобильный.
- Взаимодействие с оверлей-блокировкой (веха 2): не конфликтуют ли.
- Влияние на батарею постоянно активного VPN.
