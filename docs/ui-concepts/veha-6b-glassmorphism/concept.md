# Дизайн-концепция Glassmorphism (Веха 6Б)

Данная концепция переносит стиль **Glassmorphism** из предоставленного референса в приложение **KidGuard**. Основной упор сделан на глубокие градиенты, эффект матового стекла с блюром и мягкие неоновые свечения.

## 🎨 Визуальный язык (Общее)

*   **Иконки:** **СТРОГО ЗАПРЕЩЕНО** использование стандартных эмодзи. Используются нативные SVG-иконки (Vector Assets) в стиле Line/Fill.
*   **Стеклянные карточки (Glass Cards):** 
    *   Эффект достигается через `Modifier.blur()` (backdrop filter) и полупрозрачные фоны.
    *   Обязательная тонкая светлая кайма (border) для имитации грани стекла.
*   **Типографика:** Шрифт Inter или системный, с акцентом на жирность заголовков и читаемость на полупрозрачных фонах.

---

## 🌙 Тёмная тема (Deep Teal Glass)

*   **Фон:** Глубокий радиальный градиент. 
    *   *Верх:* `#0E1C21` (DarkBackground).
    *   *Нижние свечения:* `#04312B` (DarkOnPrimary).
*   **Стеклянные карточки:** `rgba(23, 40, 46, 0.4)` (DarkSurfaceContainer с прозрачностью).
*   **Акцентный цвет:** `#4FC3B0` (DarkPrimary) — бирюзовое свечение.
*   **Доп. акцент (Бонусы):** `#E0A73C` (DarkTertiary) — золотой.
*   **Текст:** `#E6EEF0` (DarkOnSurface).

---

## ☀️ Светлая тема (Light Teal Glass)

*   **Фон:** Светлый пастельный градиент. 
    *   *Основной:* `#EEF4F6` (LightBackground).
    *   *Свечения:* `#FFFFFF` (белые блики сверху).
*   **Стеклянные карточки:** `rgba(220, 234, 239, 0.7)` (LightSurfaceContainer с прозрачностью).
*   **Акцентный цвет:** `#2E6B7E` (LightPrimary) — глубокий сине-зеленый.
*   **Доп. акцент (Бонусы):** `#B07D1E` (LightTertiary) — золото.
*   **Текст:** `#16232A` (LightOnSurface).
*   **Объем:** Использование мягких теней (`box-shadow`) для компенсации низкой контрастности светлого стекла.

---

## 📱 Описание экранов

1.  **Dashboard (Родитель):** Круговой SVG-таймер в центре. Стеклянные кнопки бонусов (`+15`, `+30`, `★`). Список приложений в виде парящих карточек с индикаторами расхода.
2.  **Детский экран «Сегодня»:** Огромный центральный таймер. Сетка 2x2 для быстрого доступа к правилам. Нижняя плашка «Защищено KidGuard» с иконкой щита.
3.  **Статистика:** Бар-чарт с неоновой подсветкой активного дня. Список популярных приложений с процентами.
4.  **Всегда доступные:** Поле поиска с блюром. Стеклянные карточки с неоновыми переключателями (Toggle), светящимися в состоянии "ON".

---

## 🛠 Техническая реализация (Compose)

*   **Theme:** Создание `GlassColorScheme`, расширяющей стандартный Material 3.
*   **Components:** 
    *   `GlassCard`: Компонент с `Modifier.blur()` и кастомным `border`.
    *   `NeonProgress`: Круговой индикатор на основе `drawBehind` и `sweepGradient`.
    *   `GlassToggle`: Кастомный Switch с эффектом свечения.
*   **Graphics:** Все иконки через `ImageVector`.
